🚀 Job Application Tracker



📌 Description


Job Application Tracker is a simple web application that helps users manage their job applications efficiently. Users can track application status and organize them into different categories with an interactive UI.


🛠️ Technologies Used


HTML

Tailwind CSS

JavaScript (DOM API, Template Strings)



✨ Features


📂 Toggle system for All, Interview, and Rejected tabs

⚡ Dynamic rendering using JavaScript template strings

🎯 DOM manipulation to update UI based on user actions
